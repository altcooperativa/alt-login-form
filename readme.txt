WP Plugin to show the builtin WP login form in page/post content.

== Description ==

This plugin provides a shortcode with optional parameters:

[alt_login_form redirect="" form_id="" label_username=""
label_password="" label_log_in="" label_log_out="" label_lost_password=""
id_username="" id_password="" id_remember=""]

In case they are not needed, do not leave them empty, simply do not include
them.
